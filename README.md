# 可视化界面设计器

你可以用这个软件绘制你想要的程序界面，然后该软件会基于 EasyX，生成绘制这个界面的画图代码。随后你可以将这段绘图代码插入到你的项目中。

通过该程序可视化地设计程序界面，你就不需要自己在代码中使用坐标绘制程序界面了。
